# Entropy House

A cinematic, browser-based physics escape room. Solve four real-science puzzles to restore the arrow of time and escape the house.

## Play online

**[Launch Entropy House](https://anomalyaura17.github.io/entropy-house/)**

## Run locally

```bash
npm run dev
```

Then open `http://localhost:5173`.

Progress and hint use are saved locally in the browser.
